# FOH Academy Student Platform

This is the student portal for FOH Academy, built with Next.js 14, Tailwind CSS, and Firebase.

## Features
- **Modules & Lessons**: Comprehensive curriculum for Audio and Light.
- **Progress Tracking**: Track module completion.
- **Homework Submission**: Upload PDF assignments.
- **Booking System**: Reserve spots for "Praktijkdagen" (Live Training).

## Tech Stack
- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS
- **Backend/Db**: Firebase (Auth, Firestore, Storage)
- **Deployment**: Vercel

## Recent Updates
- **Curriculum Overhaul**: Transformed "Weeks" into "Modules".
- **Booking System**: Added `/booking` page.
